About

Online video game streaming is a fast growing market with a huge enthusiastic community. This website aims to give an insight in the development of this market providing a tool to measure pupularity and community interests. The provided information hopefully might prove to be of value for game developers and publishers measuring the success of their game and of course for the community itself.

This site is not an official twitch product. The data seen on this site is no indication for the quality or rating of a game but just for the popularity in the video game streaming community.

This project consists of three components working together to deliver you this service. All of them are open source:

  • Collector: collects data from twitch in regular intervals by querying the twitch api
  • Api: service that provides the collected data for the client
  • Client: this page a single page application fetching the needed data from the api

Browser compatibility testing is done via BrowserStack, who kindly provided a free plan for this open source project.

General feedback, feature requests and bug reports are very welcome and can be done using the public trello board or via the provided contact information.
